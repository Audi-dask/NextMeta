## ✨ 功能概览

- **安全 SQL 查询工作台**: 支持多数据源切换、SQL 语法高亮、相对表名书写、双击表快速预览前 20 行、结果导出以及库表级容量统计。
- **SQL 审计与规则引擎**: 集成规则引擎，对 SQL 进行语法与风险检查，可配置阻断 / 警告级规则，并在审计日志中记录触发详情。
- **工单流转与审批**: 支持查询 / 变更 / 导出工单，内置审批流、Explain 预检、风险确认与执行结果归档。
- **团队与权限 (RBAC + LDAP)**: 基于组 (Group) 的数据源访问与审批权限管控，支持 LDAP 组自动继承，显著减少手工授权成本。
- **数据脱敏与隐私安全**: 按数据源 / 库表 / 字段维度配置脱敏规则（AST 血缘分析等），防止敏感数据在查询与导出过程中被直接暴露。
- **性能安全与资源保护**: 配置全局 LIMIT、DDL/DML 执行超时与 Explain 阈值，从多维度限制大查询与高风险语句，避免烂SQL拖垮数据库实例。
- **系统设置与通知**: 集中配置全局限制、安全阈值及模版化 Webhook 通知，方便接入企业自有告警 / 通知渠道。

### 解决痛点问题

- **多渠道通知集成**：对接了国内主流飞书、钉钉、企业微信 Robot，自定义通知模版与告警内容，满足企业内部多场景通知需求。
- **随心所欲脱敏**：列级 / 字段级任意组合脱敏规则配置，支持自定义掩码方式与展示策略，并通过规则引擎防绕过，避免通过拼接 / 别名等方式绕开脱敏。
- **库表容量一目了然**：按库 / 表统计存储占用大小，方便快速识别大表与空间“黑洞”，为容量治理与归档提供依据。
- **个人SQL 收藏夹**：支持常用 SQL 片段一键收藏与分组管理，在查询工作台中快速复用、快速执行，减少重复敲命令。
- **拒绝“孤儿 SQL”**：解决执行过程中的超时、假死与“孤儿 SQL”问题，对每条 SQL 做全流程审计，在审计 / 慢 SQL 日志里都能精确追溯到执行人。
- **更直观的权限分配**：基于团队 / 组的权限视图，简化授权与回收；对接 LDAP Group 后自动继承权限，日常新增成员基本做到“免授权”接入。

## 🛠 后续计划（待开发）

- **多数据库类型扩展**：在现有能力基础上，逐步支持更多主流关系型与非关系型数据库（如 PostgreSQL、Oracle、MongoDB、Redis、ES 等），实现多类型数据库在同一平台上的统一审核与管理。
- **自动化执行与灵活工作流**：在测试环境或满足特定规则时支持自动执行工单，并提供会签 / 或签 / 条件分支等更灵活的审批流配置。
- **批量处理与工单分组**：支持将多条 DDL/DML 工单按“版本”或“发布任务”打包分组，提供批量执行与统一回溯能力。
- **可扩展与动态化审计规则**：提供更易用的规则配置与插件机制，支持通过配置或脚本自定义 SQL 检测规则，并动态调整规则级别与生效范围。
- **移动端与即时通讯深度集成**：优化移动端 H5 / 小程序体验，并与飞书、企业微信、钉钉等 IM 深度集成，支持在消息卡片中直接完成审批操作。
